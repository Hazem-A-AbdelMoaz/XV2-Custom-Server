The XV2Patcher UI extensions currently implement the following:

- Pause menu mod with options to hide the battle interface, dump auto-generated portraits and take control of cpu allies.

Instructions:

Merge the data folder with the one of the game to enable the XV2Patcher UI Extensions.
Each setting for each functionality must be enabled in the xv2patcher.ini setting.

By default, the Toggle UI and Dump portrait are enabled, and the Take control is disabled, modify the xv2patcher.ini to customize that to your liking.
